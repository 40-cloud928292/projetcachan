Structure, calibration de capteurs et tests en boucle fermée sur notre plateforme Pololu 3Pi+.

Synthèse du projet

Nous sommes un groupe de trois étudiants de l'IUT de Vélizy. Ce site centralise la documentation de notre projet d'automatisme et de programmation embarquée, basé sur le cahier des charges technique du concours de Cachan.

L'objectif critique est de configurer la matrice de réflectance infrarouge de la base Pololu 3Pi+ et d'implémenter des algorithmes de contrôle PID pour garantir un suivi de trajectoire à haute vitesse sans décrochage.

PROGRAMMES EMBARQUÉS

PID & Suivi Simple (P21)

Contrôleur Proportionnel-Dérivé de base. Kp=0.08, Kd=0.04.

main.c 
void loop() {
  sensors.readCalibrated(s);
  int error;

  if (s[2] > 500) {
    error = (s[3] - s[1]); // Centré
  } else {
    error = (s[4] + s[3]) - (s[0] + s[1]); // Déviation
  }

  if (abs(error - lastError) > 2000) error = lastError;

  int correction = (error * Kp) + ((error - lastError) * Kd);
  lastError = error;

  int leftSpeed = constrain(baseSpeed + correction, -150, 200);
  int rightSpeed = constrain(baseSpeed - correction, -150, 200);
  
  motors.setSpeeds(leftSpeed, rightSpeed);
}

bool detectLeft = (s[0] > MARKER_THRESHOLD);
bool detectRight = (s[4] > MARKER_THRESHOLD);

if ((detectLeft || detectRight) && !onMarker) {
    onMarker = true;
    crossingCount++;
    debounceCount = DEBOUNCE_LOOPS;

    switch (crossingCount) {
    case 1:  baseSpeed = SPEED_SLOW;   break;
    case 2:  baseSpeed = SPEED_FAST;   break;
    case 4:  baseSpeed = SPEED_MEDIUM; break;
