GEII Vélizy | Concours de Robotique de Cachan

Présentation du Projet

Ce site présente les travaux de développement réalisés par notre équipe de trois étudiants en Génie Électrique et Informatique Industrielle (GEII) de l'IUT de Vélizy dans le cadre du concours de Cachan. Ce concours évalue les performances d'architectures robotiques autonomes sur des parcours spécifiques.

Notre approche repose sur l'exploitation approfondie, l'étalonnage et la programmation logicielle d'un robot mobile pré-assemblé Pololu 3Pi+, configuré pour maximiser la vitesse de traitement des données et la précision des trajectoires.

PARCOURS DE COMPÉTITION

📍 Parcours 3 - Détection de Tirés (Dashes)

Algorithme de suivi de ligne avec détection de 4 tirés transversaux isolés (repères noirs perpendiculaires à la trajectoire). Le robot s'arrête immédiatement après franchissement du 4ème tiré. Utilise un filtre anti-rebond pour éviter les faux comptages et une régulation proportionnelle simple (Kp = 0.06).

#include <Pololu3piPlus32U4.h>
using namespace Pololu3piPlus32U4;

LineSensors sensors;
Motors motors;

uint16_t s[5];
int baseSpeed = 110;
float Kp = 0.06;
int lastError = 0;

// --- Arrêt fin de piste ---
int noLineCount = 0;
const int NO_LINE_THRESHOLD = 25;
const int LINE_DETECT_MIN = 200;

// --- Compteur de tirés ---
int crossingCount = 0;
const int MAX_CROSSINGS = 4;
bool onDash = false;
bool waitingToStop = false;
int debounceCount = 0;
const int DEBOUNCE_LOOPS = 80;

#define DASH_SENSOR    s[4]
#define DASH_NEIGHBOR  s[3]
const int DASH_THRESHOLD = 500;

void setup()
{
  delay(1000);
  for (int i = 0; i < 80; i++)
  {
    motors.setSpeeds((i < 40) ? 60 : -60, (i < 40) ? -60 : 60);
    sensors.calibrate();
  }
  motors.setSpeeds(0, 0);
}

void loop()
{
  sensors.readCalibrated(s);

  // --- Arrêt fin de piste ---
  int maxVal = 0;
  for (int i = 0; i < 5; i++)
    if (s[i] > maxVal) maxVal = s[i];

  if (maxVal < LINE_DETECT_MIN)
  {
    noLineCount++;
    if (noLineCount >= NO_LINE_THRESHOLD)
    {
      motors.setSpeeds(0, 0);
      while (1);
    }
  }
  else noLineCount = 0;

  // --- Décompte anti-rebond ---
  if (debounceCount > 0)
  {
    debounceCount--;
  }
  else
  {
    // --- Détection tiré isolé ---
    bool dashAlone = (DASH_SENSOR > DASH_THRESHOLD) && (DASH_NEIGHBOR < DASH_THRESHOLD);

    if (dashAlone && !onDash)
    {
      onDash = true;
      crossingCount++;
      debounceCount = DEBOUNCE_LOOPS;

      if (crossingCount >= MAX_CROSSINGS)
        waitingToStop = true;
    }

    if (!dashAlone && onDash)
    {
      onDash = false;

      if (waitingToStop)
      {
        motors.setSpeeds(0, 0);
        while (1);
      }
    }
  }

  // --- Suivi de ligne ---
  int error = (s[4] + s[3]) - (s[0] + s[1]);
  if (s[2] > (s[1] + s[3]) / 2)
    error = (s[3] - s[1]);
  if (abs(error - lastError) > 1500)
    error = lastError;

  lastError = error;

  int correction = error * Kp;
  int leftSpeed  = baseSpeed + correction;
  int rightSpeed = baseSpeed - correction;
  leftSpeed  = constrain(leftSpeed, -150, 200);
  rightSpeed = constrain(rightSpeed, -150, 200);
  motors.setSpeeds(leftSpeed, rightSpeed);
}

📍 Parcours 9 - Détection Latérale + Virage

Suivi de ligne avec détection de 7 marques latérales (tirés sur les côtés gauche ou droite). À la détection de la 7ème marque, le robot effectue un virage à 90° avant de continuer. Implémente un contrôle PD (avec dérivée) pour une meilleure stabilité sur les courbes. Kp = 0.04, Kd = 0.08.

#include <Pololu3piPlus32U4.h>
using namespace Pololu3piPlus32U4;

LineSensors sensors;
Motors motors;

uint16_t s[5];
int baseSpeed = 110;
float Kp = 0.04;
float Kd = 0.08;
int lastError = 0;

// --- Arrêt fin de piste ---
int noLineCount = 0;
const int NO_LINE_THRESHOLD = 25;
const int LINE_DETECT_MIN = 200;

// --- Compteur lateral ---
int sideCount = 0;
const int TURN_AT = 7;
bool onMark = false;
int debounceCount = 0;
const int DEBOUNCE_LOOPS = 80;
bool turnDone = false;

const int SIDE_THRESHOLD = 500;

void setup()
{
  delay(1000);
  for (int i = 0; i < 80; i++)
  {
    motors.setSpeeds((i < 40) ? 60 : -60, (i < 40) ? -60 : 60);
    sensors.calibrate();
  }
  motors.setSpeeds(0, 0);
}

void loop()
{
  sensors.readCalibrated(s);

  // --- Arrêt fin de piste ---
  int maxVal = 0;
  for (int i = 0; i < 5; i++)
  {
    if (s[i] > maxVal) maxVal = s[i];
  }

  if (maxVal < LINE_DETECT_MIN)
  {
    noLineCount++;
    if (noLineCount >= NO_LINE_THRESHOLD)
    {
      motors.setSpeeds(0, 0);
      while (1);
    }
  }
  else
  {
    noLineCount = 0;
  }

  // --- Détection marque gauche OU droite ---
  if (debounceCount > 0)
  {
    debounceCount--;
  }
  else
  {
    bool markDetected = (s[4] > SIDE_THRESHOLD) || (s[0] > SIDE_THRESHOLD);

    if (markDetected && !onMark)
    {
      onMark = true;
      sideCount++;
      debounceCount = DEBOUNCE_LOOPS;

      if (sideCount == TURN_AT && !turnDone)
      {
        turnDone = true;
        motors.setSpeeds(0, 0);
        delay(200);

        motors.setSpeeds(50, -50);
        delay(375);
        motors.setSpeeds(0, 0);
        delay(100);

        lastError = 0;
      }
    }

    if (!markDetected && onMark)
    {
      onMark = false;
    }
  }

  // --- Suivi de ligne PD ---
  int error = (s[4] + s[3]) - (s[0] + s[1]);
  if (s[2] > (s[1] + s[3]) / 2)
  {
    error = (s[3] - s[1]);
  }
  if (abs(error - lastError) > 1500)
  {
    error = lastError;
  }

  int correction = (error * Kp) + ((error - lastError) * Kd);
  lastError = error;

  int leftSpeed = baseSpeed + correction;
  int rightSpeed = baseSpeed - correction;

  if (leftSpeed > 200) leftSpeed = 200;
  if (leftSpeed < -150) leftSpeed = -150;
  if (rightSpeed > 200) rightSpeed = 200;
  if (rightSpeed < -150) rightSpeed = -150;

  motors.setSpeeds(leftSpeed, rightSpeed);
}

📍 Parcours 13 - Machine d'États Complexe

Implémentation multi-états pour gérer des transitions de parcours complexes. État 0 : suivi initial + 7 marques + virage 90°. État 1 : continuation avec détection fin de piste et retournement 180°. État 2 : deuxième phase avec 3 marques + virage 90°. État 3 : arrêt final. Démontre une gestion avancée de la navigation autonome.

#include <Pololu3piPlus32U4.h>
using namespace Pololu3piPlus32U4;

LineSensors sensors;
Motors motors;

uint16_t s[5];
int baseSpeed = 110;
float Kp = 0.04;
float Kd = 0.08;
int lastError = 0;

int state = 0;
int noLineCount = 0;
const int NO_LINE_THRESHOLD = 50;
const int LINE_DETECT_MIN = 200;

int sideCount = 0;
bool onMark = false;
int debounceCount = 0;
const int DEBOUNCE_LOOPS = 80;
const int SIDE_THRESHOLD = 500;

void turn90Right()
{
  motors.setSpeeds(0, 0);
  delay(200);
  motors.setSpeeds(50, -50);
  delay(375);
  motors.setSpeeds(0, 0);
  delay(100);
  lastError = 0;
}

void turn180()
{
  motors.setSpeeds(0, 0);
  delay(200);
  motors.setSpeeds(55, -55);
  delay(750);
  motors.setSpeeds(0, 0);
  delay(100);
  lastError = 0;
}

void setup()
{
  delay(1000);
  for (int i = 0; i < 80; i++)
  {
    motors.setSpeeds((i < 40) ? 60 : -60, (i < 40) ? -60 : 60);
    sensors.calibrate();
  }
  motors.setSpeeds(0, 0);
}

void loop()
{
  sensors.readCalibrated(s);

  int maxVal = 0;
  for (int i = 0; i < 5; i++)
  {
    if (s[i] > maxVal) maxVal = s[i];
  }

  if (maxVal < LINE_DETECT_MIN)
  {
    noLineCount++;
    if (noLineCount >= NO_LINE_THRESHOLD)
    {
      if (state == 1)
      {
        motors.setSpeeds(0, 0);
        delay(500);
        turn180();

        unsigned long startTime = millis();
        while (millis() - startTime < 1000)
        {
          sensors.readCalibrated(s);
          motors.setSpeeds(110, 110);
        }

        sideCount = 0;
        onMark = false;
        debounceCount = 0;
        noLineCount = 0;
        lastError = 0;
        state = 2;
        return;
      }
      else if (state == 3)
      {
        motors.setSpeeds(0, 0);
        while (1);
      }
      else if (state != 2)
      {
        motors.setSpeeds(0, 0);
        while (1);
      }
    }
  }
  else
  {
    noLineCount = 0;
  }

  if (debounceCount > 0)
  {
    debounceCount--;
  }
  else
  {
    bool markDetected = (s[4] > SIDE_THRESHOLD) || (s[0] > SIDE_THRESHOLD);

    if (markDetected && !onMark)
    {
      onMark = true;
      sideCount++;
      debounceCount = DEBOUNCE_LOOPS;

      if (state == 0 && sideCount == 7)
      {
        turn90Right();
        sideCount = 0;
        state = 1;
      }

      if (state == 2 && sideCount == 3)
      {
        turn90Right();
        sideCount = 0;
        state = 3;
      }
    }

    if (!markDetected && onMark)
    {
      onMark = false;
    }
  }

  int error = (s[4] + s[3]) - (s[0] + s[1]);
  if (s[2] > (s[1] + s[3]) / 2)
  {
    error = (s[3] - s[1]);
  }
  if (abs(error - lastError) > 1500)
  {
    error = lastError;
  }

  int correction = (error * Kp) + ((error - lastError) * Kd);
  lastError = error;

  int leftSpeed = baseSpeed + correction;
  int rightSpeed = baseSpeed - correction;

  if (leftSpeed > 200) leftSpeed = 200;
  if (leftSpeed < -150) leftSpeed = -150;
  if (rightSpeed > 200) rightSpeed = 200;
  if (rightSpeed < -150) rightSpeed = -150;

  motors.setSpeeds(leftSpeed, rightSpeed);
}

📍 Parcours 13 (v2) - Aller-Retour Simple

Version optimisée du parcours 13 avec machine à 2 états. Phase aller : suivi de ligne jusqu'à perte de signal. Détection automatique de fin de parcours, exécution d'un demi-tour 180°, puis phase retour. Arrêt immédiat si perte de ligne en retour. Kp = 0.08, Kd = 0.04 pour réactivité maximale.

#include <Pololu3piPlus32U4.h>
using namespace Pololu3piPlus32U4;

LineSensors sensors;
Motors motors;

uint16_t s[5];

int baseSpeed = 110;
float Kp = 0.08;
float Kd = 0.04;
int lastError = 0;

int state = 0; // 0 = aller, 1 = retour
int noLineCount = 0;
const int NO_LINE_THRESHOLD = 50;
const int LINE_DETECT_MIN = 200;

void turn180()
{
  motors.setSpeeds(0, 0);
  delay(200);
  // DEMI-TOUR (400 ms)
  motors.setSpeeds(100, -100);
  delay(400);
  motors.setSpeeds(0, 0);
  delay(200);
  // petit avance pour retrouver la ligne
  motors.setSpeeds(100, 100);
  delay(250);
  motors.setSpeeds(0, 0);
  delay(100);
  lastError = 0;
}

void stopForever()
{
  motors.setSpeeds(0, 0);
  while (true)
  {
    delay(1000);
  }
}

void setup()
{
  delay(1000);
  // calibration des capteurs
  for (int i = 0; i < 80; i++)
  {
    if (i < 40)
      motors.setSpeeds(60, -60);
    else
      motors.setSpeeds(-60, 60);
    sensors.calibrate();
  }
  motors.setSpeeds(0, 0);
  delay(500);
}

void loop()
{
  sensors.readCalibrated(s);

  int maxVal = 0;
  // recherche de la valeur max des capteurs
  for (int i = 0; i < 5; i++)
  {
    if (s[i] > maxVal)
      maxVal = s[i];
  }

  // détection perte de ligne
  if (maxVal < LINE_DETECT_MIN)
    noLineCount++;
  else
    noLineCount = 0;

  // FIN ALLER → demi-tour
  if (state == 0 && noLineCount >= NO_LINE_THRESHOLD)
  {
    motors.setSpeeds(0, 0);
    delay(300);
    turn180();
    state = 1;
    noLineCount = 0;
    return;
  }

  // FIN RETOUR → arrêt
  if (state == 1 && noLineCount >= NO_LINE_THRESHOLD)
  {
    stopForever();
  }

  // ===== suivi de ligne =====
  int error;
  if (s[2] > 500)
  {
    error = s[3] - s[1];
  }
  else
  {
    error = (s[4] + s[3]) - (s[0] + s[1]);
  }

  // filtrage des grosses erreurs
  if (abs(error - lastError) > 2000)
    error = lastError;

  int correction =
    (error * Kp) +
    ((error - lastError) * Kd);
  lastError = error;

  int leftSpeed = baseSpeed + correction;
  int rightSpeed = baseSpeed - correction;

  leftSpeed = constrain(leftSpeed, -150, 200);
  rightSpeed = constrain(rightSpeed, -150, 200);

  motors.setSpeeds(leftSpeed, rightSpeed);
}

🎓 Mission 24 - Suivi de Ligne Simple

Niveau introductif : suivi basique de ligne sans détection d'obstacles ou de marques. Le robot suit simplement la ligne noire en utilisant une régulation PD (Kp = 0.08, Kd = 0.04). Pas d'arrêt, continue indéfiniment. Utile pour calibrer et tester la boucle de contrôle.

#include <Pololu3piPlus32U4.h>
using namespace Pololu3piPlus32U4;

LineSensors sensors;
Motors motors;
uint16_t s[5];

int baseSpeed = 110;
float Kp = 0.08;
float Kd = 0.04;
int lastError = 0;

void setup()
{
  delay(1000);
  for (int i = 0; i < 80; i++)
  {
    motors.setSpeeds((i < 40) ? 60 : -60, (i < 40) ? -60 : 60);
    sensors.calibrate();
  }
  motors.setSpeeds(0, 0);
}

void loop()
{
  sensors.readCalibrated(s);
  
  // ========== SUIVI DE LIGNE SIMPLE ==========
  int error;
  
  if (s[2] > 500)
  {
    // Ligne bien centrée
    error = (s[3] - s[1]);
  }
  else
  {
    // Ligne sur les côtés
    error = (s[4] + s[3]) - (s[0] + s[1]);
  }
  
  if (abs(error - lastError) > 2000)
    error = lastError;
  
  int correction = (error * Kp) + ((error - lastError) * Kd);
  lastError = error;
  
  int leftSpeed = baseSpeed + correction;
  int rightSpeed = baseSpeed - correction;
  
  leftSpeed = constrain(leftSpeed, -150, 200);
  rightSpeed = constrain(rightSpeed, -150, 200);
  
  motors.setSpeeds(leftSpeed, rightSpeed);
}

🎓 Mission 27 - Aller-Retour avec Demi-Tour

Intermédiaire : le robot suit la ligne jusqu'à détection de fin (perte de signal). À ce moment, exécute un demi-tour (1000 ms) avant de continuer en phase retour. Pas d'arrêt programmatique, continue jusqu'à fin du parcours. Démontre la gestion de transitions d'état.

#include <Pololu3piPlus32U4.h>
using namespace Pololu3piPlus32U4;

LineSensors sensors;
Motors motors;
uint16_t s[5];

int baseSpeed = 110;
float Kp = 0.08;
float Kd = 0.04;
int lastError = 0;

int state = 0;  // 0=Aller, 1=Retour
int noLineCount = 0;
const int NO_LINE_THRESHOLD = 50;
const int LINE_DETECT_MIN = 200;

void turn180()
{
  motors.setSpeeds(0, 0);
  delay(200);
  motors.setSpeeds(100, -100);
  delay(1000);
  motors.setSpeeds(0, 0);
  delay(100);
  lastError = 0;
}

void setup()
{
  delay(1000);
  for (int i = 0; i < 80; i++)
  {
    motors.setSpeeds((i < 40) ? 60 : -60, (i < 40) ? -60 : 60);
    sensors.calibrate();
  }
  motors.setSpeeds(0, 0);
}

void loop()
{
  sensors.readCalibrated(s);
  
  int maxVal = 0;
  for (int i = 0; i < 5; i++)
  {
    if (s[i] > maxVal) maxVal = s[i];
  }
  
  // ========== ALLER ==========
  if (state == 0)
  {
    // Détecte fin de ligne
    if (maxVal < LINE_DETECT_MIN)
    {
      noLineCount++;
      if (noLineCount >= NO_LINE_THRESHOLD)
      {
        // Fin atteinte: faire demi-tour
        motors.setSpeeds(0, 0);
        delay(500);
        turn180();
        
        unsigned long startTime = millis();
        while (millis() - startTime < 800)
        {
          sensors.readCalibrated(s);
          motors.setSpeeds(110, 110);
        }
        
        noLineCount = 0;
        lastError = 0;
        state = 1;  // Passer au retour
        return;
      }
    }
    else
    {
      noLineCount = 0;
    }
  }
  
  // ========== SUIVI DE LIGNE ==========
  int error;
  
  if (s[2] > 500)
  {
    error = (s[3] - s[1]);
  }
  else
  {
    error = (s[4] + s[3]) - (s[0] + s[1]);
  }
  
  if (abs(error - lastError) > 2000)
    error = lastError;
  
  int correction = (error * Kp) + ((error - lastError) * Kd);
  lastError = error;
  
  int leftSpeed = baseSpeed + correction;
  int rightSpeed = baseSpeed - correction;
  
  leftSpeed = constrain(leftSpeed, -150, 200);
  rightSpeed = constrain(rightSpeed, -150, 200);
  
  motors.setSpeeds(leftSpeed, rightSpeed);
}

🎓 Mission 30 - Compter 3 Tours Autour d'un Point

Avancé : le robot suit une boucle fermée et compte les passages de marques (tirets). ~6 tirets par tour autour du point A. Arrêt automatique après 3 tours complétés (18 marques détectées). Démontre détection de patterns et comptage événementiel avec anti-rebond.

#include <Pololu3piPlus32U4.h>
using namespace Pololu3piPlus32U4;

LineSensors sensors;
Motors motors;
uint16_t s[5];

int baseSpeed = 110;
float Kp = 0.08;
float Kd = 0.04;
int lastError = 0;

int tourCount = 0;  // Compte les passages au point de départ
int markCount = 0;
bool onMark = false;
int debounceCount = 0;
const int DEBOUNCE_LOOPS = 150;
const int SIDE_THRESHOLD = 450;

void setup()
{
  delay(1000);
  for (int i = 0; i < 80; i++)
  {
    motors.setSpeeds((i < 40) ? 60 : -60, (i < 40) ? -60 : 60);
    sensors.calibrate();
  }
  motors.setSpeeds(0, 0);
}

void loop()
{
  sensors.readCalibrated(s);
  
  // ========== COMPTER LES PASSAGES (tirets) ==========
  if (debounceCount > 0)
  {
    debounceCount--;
  }
  else
  {
    bool markDetected = (s[4] > SIDE_THRESHOLD) || (s[0] > SIDE_THRESHOLD);
    
    if (markDetected && !onMark)
    {
      onMark = true;
      markCount++;
      debounceCount = DEBOUNCE_LOOPS;
      
      // Après 6 tirets = 1 tour (il y a ~6 tirets par tour autour de A)
      tourCount = markCount / 6;
      
      // Au 3ème tour: arrêter
      if (tourCount >= 3)
      {
        motors.setSpeeds(0, 0);
        // 3 tours complétés!
        return;
      }
    }
    
    if (!markDetected && onMark)
    {
      onMark = false;
    }
  }
  
  // ========== SUIVI DE LIGNE ==========
  int error;
  
  if (s[2] > 500)
  {
    error = (s[3] - s[1]);
  }
  else
  {
    error = (s[4] + s[3]) - (s[0] + s[1]);
  }
  
  if (abs(error - lastError) > 2000)
    error = lastError;
  
  int correction = (error * Kp) + ((error - lastError) * Kd);
  lastError = error;
  
  int leftSpeed = baseSpeed + correction;
  int rightSpeed = baseSpeed - correction;
  
  leftSpeed = constrain(leftSpeed, -150, 200);
  rightSpeed = constrain(rightSpeed, -150, 200);
  
  motors.setSpeeds(leftSpeed, rightSpeed);
}

Notes Techniques sur l'Implémentation

Progression Pédagogique :

  • Parcours 3, 5, 9, 13 : Cas de compétition avec contraintes multiples (détection de marques, virages, arrêts programmés).
  • Missions 24-31 : Progression d'entraînement (niveau intro → expert) pour maîtriser progressivement suivi de ligne, transitions, comptage événementiel.

Architecture Commune :

  • Librairie Pololu 3Pi+ : Classes LineSensors (lecture calibrée 5 capteurs IR) et Motors (PWM moteurs).
  • Boucle Temps Réel : Fréquence ~1 kHz sur AVR 20 MHz pour réactivité maximale.
  • Régulation PD : Proportionnelle-Dérivée avec filtrage pics d'erreur (seuil 1500-2000) pour stabilité trajectoire.
  • Détection Marques : Seuillage adaptatif (SIDE_THRESHOLD: 450-500) + anti-rebond (DEBOUNCE_LOOPS: 80-150).
  • Machine d'États : Transitions de phase pour aller-retour, comptage tours, arrêts conditionnels.
