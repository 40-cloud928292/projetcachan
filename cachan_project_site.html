Bienvenue dans notre projet

Nous sommes une équipe de trois étudiants en électronique et informatique industrielles (GEII) à Vélizy, passionnés par la robotique. Notre projet consiste à développer un robot autonome capable de naviguer et de réussir les épreuves du concours Cachan, l'une des plus prestigieuses compétitions nationales de robotique en France.

Notre robot Pololu 3Pi+ est une plateforme compacte et performante que nous avons programmée et optimisée pour relevé des défis robotiques complexes.

Notre Robot

Pololu 3Pi+ - Une plateforme performante et modulaire

🤖
📐 Plateforme de Base
Robot Pololu 3Pi+ : plateforme compacte à trois roues, conçue spécifiquement pour la compétition robotique et les parcours autonomes complexes.
⚙️ Système Moteur
Deux moteurs DC réducteurs haute performance assurant une accélération rapide et une stabilité directionnelle optimale sur les trajectoires courbes.
🔍 Capteurs Infrarouge
Cinq capteurs infrarouges pour détecter les lignes de sol et les obstacles. Permettent une navigation précise le long du parcours marqué.
🧠 Microcontrôleur AVR
Processeur 32-bit fonctionnant à 20 MHz avec mémoire suffisante pour les algorithmes complexes. Exécution temps réel des boucles de contrôle.
🔋 Système d'Alimentation
Batterie rechargeable optimisée pour 2 à 3 heures d'autonomie. Gestion intelligente de l'énergie pour maximiser le temps de fonctionnement en compétition.
⚡ Électronique Additionnelle
Circuits de conditionnement des signaux, amplificateurs pour capteurs, et drivers de moteur pour une intégration optimale de tous les composants.

Épreuves du Concours

Cachan - Compétition Nationale de Robotique

Le concours Cachan propose plusieurs épreuves testant la performance et la robustesse des robots

🏁

Sprint en Ligne Droite

Le robot doit parcourir une distance définie en ligne droite le plus rapidement possible. Teste l'accélération et la stabilité directionnelle.

Temps chronométré
🌀

Parcours Sinueux

Navigation sur un tracé avec virages et courbes. Le robot doit suivre précisément le chemin sans dévier. Teste la réactivité et la cinématique.

Précision et temps
🚧

Franchissement d'Obstacles

Le robot doit détecter et contourner des obstacles disposés sur le parcours. Teste l'autonomie et les capacités de détection.

Détection et contournement
🔌

Fiabilité Énergétique

Le robot doit accomplir toutes ses missions avec une seule charge de batterie. Teste l'optimisation énergétique du système.

Endurance et efficacité
🎯

Précision de Positionnement

Le robot doit atteindre des positions exactes définies sur l'aire de compétition. Teste la calibration et la précision des capteurs.

Localisation autonome
⏱️

Parcours Complet Chronométré

Le robot doit accomplir un parcours complexe incluant tous les défis précédents en un temps limité. L'épreuve finale décisive.

Performance globale

Algorithmes et code embarqué du robot

Découvrez les programmes et algorithmes qui font fonctionner notre robot Pololu 3Pi+

🔄 Firmware Principal 75% IA

Code embarqué complet du microcontrôleur AVR. Gère la boucle principale, les interruptions, les contrôles moteur et la lecture des capteurs en temps réel.

📥 Télécharger le code
⚡ 75% de ce code a été optimisé/généré avec assistance IA

🧭 Algoritme de Navigation 45% IA

Algorithme de suivi de ligne développé pour naviguer précisément le long du parcours. Implémente des boucles PID pour corriger la trajectoire en fonction des capteurs.

📥 Télécharger le code
⚡ 45% de ce code provient de suggestions et optimisations IA

🤖 Détection d'Obstacles 80% IA

Système de détection et de contournement automatique des obstacles. Utilise les capteurs infrarouges pour identifier les barrières et ajuste la trajectoire.

📥 Télécharger le code
⚡ 80% de ce module a été généré et affiné avec IA

⚙️ Contrôle Moteur (PWM) 10% IA

Gestion des moteurs via PWM (Pulse Width Modulation). Code bas niveau pour contrôler la vitesse et la direction des deux moteurs du robot.

📥 Télécharger le code
⚡ 10% de ce code provient de suggestions IA, principalement du code manuel

🔍 Calibration des Capteurs 60% IA

Utilitaire de calibration automatique des capteurs infrarouges. Ajuste les seuils de détection en fonction de l'éclairage et de l'environnement.

📥 Télécharger le code
⚡ 60% de ce programme a été créé avec assistance IA

⏱️ Gestion d'Énergie 35% IA

Système de gestion de l'alimentation pour optimiser la consommation énergétique. Surveille la batterie et ajuste les paramètres pour maximiser l'autonomie.

📥 Télécharger le code
⚡ 35% de ce système provient d'optimisations IA

📌 Note sur l'utilisation de l'IA

Les pourcentages d'IA indiquent le niveau d'assistance d'une IA générative (ChatGPT, Claude, etc.) dans la création, optimisation et débuggage du code. Le développement, les tests et la validation finale restent sous responsabilité complète de notre équipe.
